# Mytheos - AI-Powered Debugging Multi-Agent System

Automatically debug code, identify root causes, generate fixes, and create pull requests using AI agent collaboration.

## Features

- **Multi-Agent Architecture**: Specialized agents for each debugging phase
- **Automatic Bug Detection**: Parse error logs and identify issues
- **Root Cause Analysis**: Deep analysis to find true causes
- **Fix Generation**: AI-generated code fixes
- **Test Generation**: Comprehensive test suites
- **Pull Request Ready**: Auto-generated PRs with full context

## Installation

```bash
# Clone repository
git clone https://github.com/ChiragSindhu/mytheos_microsoftbuildaihackathon.git
cd mytheos

# Setup (creates venv, installs dependencies)
chmod +x scripts/setup.sh
./scripts/setup.sh

# Activate environment
source venv/bin/activate  # Linux/Mac
# or
venv\Scripts\activate  # Windows
